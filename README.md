# Documentação do sistema de cadastro de clientes

## Descrição
O repositório consiste em um sistema para cadastro de clientes, de pessoa física ou jurídica, por meio de linha da comandos

## Features
Com o sistema é possível cadastrar pessoa fisica ou jurídica. O sistema pede o CPF ou CPNP, bem como os principais dados como: nome, endereço, rendimento. De forma simplificada, o sistema também calcula o imposto a ser pago.

## Tecnologias utilizadas
Para desenvolvimento do sistema foi utilizada a linguagem de programação C#.

## Organização do projeto
O projeto foi organizado de forma a ser intuitivo e de facil usabilidade com o usuário.

## Pré-requisitos de instalção
Ter o .NET instalado.


## Execução da aplicação
Para execução da aplicação é necessário digitar ".net" na linha de comando e ir respondendo as solicitações do sistema.

# Erros comuns
Os erros que podem apresentar até o momento são alertados dentro do próprio sistema, como por exemplo CNPJ digitado fora do padrão.
